# ML Engineer Nanodegree

Projects completed as part of Udacity's Machine Learning Engineer Nanodegree program.

## Projects

- **[boston_housing](https://github.com/burnssa/boston_housing)** - Predicting housing prices using decision tree regression
- **[student_intervention](https://github.com/burnssa/student_intervention)** - Classifying students at risk of dropout using SVM, AdaBoost, and K-nearest neighbors
- **[creating_customer_segments](https://github.com/burnssa/creating_customer_segments)** - Identifying customer segments using unsupervised learning and clustering techniques
- **[smartcab_reinforcement_learning](https://github.com/burnssa/smartcab_reinforcement_learning)** - Training a reinforcement learning agent to navigate a grid environment and obey traffic rules
- **[street_view_digit_recognition](https://github.com/burnssa/street_view_digit_recognition)** - Training a neural network to recognize and convert street address digits from images to numeric values
